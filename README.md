# Braneworld-EMRI

This file contains the flux computation for eccentric extreme mass-ratio inspirals with braneworld geometry. In the notebook, a parameter $\delta$ implies Dudley-Finley with \delta = 0 (where Q contribution comes from metric function \Delta(r,Q)) and modified Teukolsky contribution \delta = 1 (where Q contribution comes from the non-Ricci flat nature of the background geometry and metric function \Delta(r,Q)).

This can further be use to construct a two-dimensional grid in semi-latus rectum (p) and eccentricity (e) for fixed values of tidal charge (Q).
